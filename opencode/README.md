# OpenCode Configuration

This directory contains OpenCode configuration files for the dotfiles repository.

## Files

- `config.toml` - Main OpenCode configuration
- `README.md` - This file

## Installation

OpenCode is automatically linked when running the main installation script:
```bash
./install.sh -p opencode
```

Or install all packages:
```bash
./install.sh
```

## Usage

OpenCode will automatically detect and use the configuration in this directory when working within this repository.
